DESIGN SYSTEM v1.0 · ABRIL 2026

ComunidadConnect.
Un lenguaje visual para tres roles.

Sistema de diseño unificado para administradores, conserjes y residentes. Tokens, componentes y patrones construidos sobre el dark theme existente — refinado, consistente y listo para escalar.

Base: Dark theme violeta Fuentes: Bricolage Grotesque + Geist Stack: Next.js + Tailwind
