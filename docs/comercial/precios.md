# Precios propuestos

Valores iniciales para venta consultiva en Chile. Ajustar antes de enviar una propuesta formal.

## Plan Esencial

- Para comunidades hasta 80 unidades.
- $49.900 CLP base mensual.
- $490 CLP por unidad activa al mes.
- Incluye residentes, conserjeria, comunicaciones, reservas, marketplace, directorio, cargas masivas y soporte estandar.

## Plan Pro

- Para comunidades de 81 a 250 unidades.
- $89.900 CLP base mensual.
- $790 CLP por unidad activa al mes.
- Incluye todo Esencial mas CoCo IA, casos inteligentes, cursos IA, dashboard operativo, control hidrico, votaciones y reportes.

## Plan Enterprise

- Para administradoras, multiples comunidades o integraciones especiales.
- Desde $199.000 CLP base mensual.
- $990 CLP por unidad activa al mes.
- Incluye multi-comunidad, SLA prioritario, integraciones a medida, capacitacion extendida y mesa de implementacion.

## Onboarding

- Implementacion inicial: $149.000 CLP por comunidad.
- Incluye carga de unidades/residentes, configuracion de roles, capacitacion y validacion.
- Se puede bonificar con contrato anual.

## Add-ons

- WhatsApp: costo de proveedor + $15.000 CLP mensual de administracion.
- Facturacion/boletas: costo proveedor + configuracion unica desde $99.000 CLP.
- Integracion IoT/agua: cotizacion segun medidores y fuente de datos.
- Dominio personalizado: $25.000 CLP mensual si Convive Connect administra DNS y soporte.

## Descuento anual

- 10% de descuento por pago anual anticipado.
- Setup bonificado para contratos anuales Pro o Enterprise.
