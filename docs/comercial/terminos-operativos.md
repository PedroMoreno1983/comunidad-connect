# Terminos operativos

Documento base para venta y operacion. Requiere revision legal antes de publicarse como terminos finales.

## Uso aceptable

La plataforma debe usarse para gestion comunitaria, comunicaciones, reservas, solicitudes, visitas, marketplace vecinal, reportes y automatizaciones autorizadas.

Esta prohibido usarla para spam, acoso, discriminacion, publicacion de datos sensibles sin autorizacion, suplantacion o decisiones automatizadas de alto impacto sin revision humana.

## Datos

El cliente es responsable de la legitimidad y calidad de los datos cargados. Convive Connect mantiene controles de acceso, logs y segregacion por comunidad.

Los datos de una comunidad no deben ser visibles para otra comunidad.

## IA

CoCo IA puede resumir, clasificar, responder, recomendar acciones y ayudar a encontrar informacion. Las respuestas deben considerarse asistencia operativa, no asesoria legal definitiva.

Para materias de copropiedad, seguridad de la informacion, reglamentos, multas, datos personales o decisiones formales, la administracion debe validar antes de ejecutar.

## Envios

Los envios por correo, WhatsApp o notificaciones deben respetar consentimiento, finalidad y reglas internas del cliente. Los canales reales deben probarse primero con destinatarios internos autorizados.

## Seguridad

Cada usuario debe proteger sus credenciales. El administrador debe retirar accesos cuando una persona deja de pertenecer a la comunidad o cambia de rol.

## Cambios

Convive Connect puede actualizar funcionalidades para mejorar seguridad, estabilidad y experiencia. Cambios materiales de precio o alcance se comunicaran con aviso razonable.
